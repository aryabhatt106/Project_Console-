How to Set up Visual code Studio

1. Download Visual code Studio on your PC. Click to Download

2. Make a file named anything you like. Ex. Javascript, Visual Code Studio etc

3. Make a Folder inside the File you had made

4. Open Visual Code Studio and and press File

5. After pressing File press Open folder and choose the folder you made

6. Make files index.html, style.css, main.js

7. CODE!!

The Console Log

The Console log is used to debugg your code.To open the browser console, right-click on the page and select Inspect, and then click Console.
